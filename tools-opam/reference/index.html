    "Here" switch & coswitch operations

    • @opam//here

    • @opam//here:clean - delete coswitch but retain OPAM here-switch

    • @opam//here:clone - create here coswitch by cloning shared OPAM switch

    • @opam//here:create - create here switch and coswitch

    • @opam//here:export - export here coswitch metadata

    • @opam//here:expunge - delete entire here switch and coswitch

    • @opam//here:import - import here-coswitch from exported metadata

    • @opam//here:list - print listing of coswitch config

    • @opam//here:refresh - update here coswitch from here switch

    • @opam//here:reinit - re-initialize here-coswitch

    • @opam//here:set - make here-coswitch the effective coswitch for builds
