Regenerate `here` coswitch (`.obazl.d/opam/here`) from OPAM `here`
switch (`.opam`).

This command should be run whenever the OPAM `here` switch changes
(for example, after running `@opam//here/opam:install` or
`@opam//here/opam:remove`).
